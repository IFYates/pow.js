Features / Interactivity

By using a function expression, any element event can be bound to with a on(event)="{{ function }}" attribute. 

    <button pow onclick="clicked">
        Click me
    </button>
When invoked, the bound function will receive the current context data and the root data. 

    function clicked(data, root) {
        // this: The element that was clicked
        // data: The context data
        // root: The root data
    }
