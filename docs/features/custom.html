Custom elements

Custom elements are a way to use more succinctly, by using the template id as the element name.

The following lines of code are identical ways of using the my-component template:

<template pow template="my-component"></template>
<pow:my-component></pow:my-component>

Custom elements are replaced with the parsed contents of the same template.

Subcontent also works the same way as templates. 

<pow:my-component>
    <template>Title</template>
</pow:my-component>

        
<template id="my-component">
    <div><param /></div>
</template>

    


    

        Because a custom element is a <template> element with a pow binding, other  binding attributes are directly supported.
