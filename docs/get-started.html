Getting started

The quickest way to start using is to directly import the latest distribution build from the publish site.

The following can be saved to a local file and opened in your browser: 

<head>
    <script type="module">
    import pow from 'https://ifyates.github.io/pow.js/latest/pow.min.js'
    const data = { text: 'Hello, world!' }
    
    </script>
</head>
<body>
    <div>
        
    </div>
</body>

    


    

        Check out a complete example here: https://jsfiddle.net/IFYates/qtngjxbu
