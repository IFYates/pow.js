Getting started

The quickest way to get started is to use the latest version of from the publish site.
This can be imported directly to immediately build a functional page.

The following can be saved to a local file and opened in your browser: 

<head>
    <script type="module">
        import pow from 'https://ifyates.github.io/pow.js/latest/pow.min.js'
        const data = { text: 'Hello, world!' }
        pow.apply(document.body, data)
    </script>
</head>
<body>
    <div>{{ text }}</div>
</body>

Check out a complete example here: https://jsfiddle.net/IFYates/qtngjxbu
