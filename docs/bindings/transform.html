Transformation

Syntax: transform="function"

The function is invoked after all processing has completed for the element, and receives the current element as an argument and the current context as arguments. 

{ upperCase: (el, context) => el.innerHTML = context.$path + ' :: ' + el.innerHTML.toUpperCase() }
<div>
    <template pow transform="upperCase">
        Hello, world!
    </template>
</div>

    


    

        The value of function must return a function. Any other
        result
        will be ignored, including expressions.
    

    

        
<div>
    <template pow transform="'Removed'">
        Non-function expression
    </template>
</div>
<div>
    <template pow transform="{{ '(el) => el.innerHTML = `Removed`' }}">
        Expression function
    </template>
</div>
