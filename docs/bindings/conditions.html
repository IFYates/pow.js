Bindings / Conditions

if="condition" / ifnot="condition"

The if and ifnot attributes can be used to conditionally render any bound element.

If the condition evaluates to a truthy value, the element is rendered, otherwise it is removed. 

<div pow if="true">
    First block
</div>
<div pow if="false">
    Second block
</div>








    else
    / else-if="condition"
    / else-ifnot="condition"




    If a conditional element has been processed, the following sibling elements can make use of the
    else* attributes to further conditionally render an element.




    
<div pow if="false">
    First block
</div>
<div pow else>
    Second block
</div>

    






    This can be chained for as many elements as needed.




    
{ value: 1 }

    
<div pow if="value == 1">
    One
</div>
<div pow else-if="value == 2">
    Two
</div>
<div pow else-if="value == 3">
    Three
</div>
<div pow else-ifnot="value > 0">
    Negative
</div>
<div pow else>
    More than 3
</div>
